# odin-recipes
## Introduction
This project is a simple recipe website built as part of The Odin Project curriculum. 
It demonstrates basic HTML skills including creating web pages, adding links, images, and lists.

## Skills Demonstrated
Upon completing this project, I will have demonstrated:
- Creating multiple HTML pages
- Linking between pages using anchor tags
- Adding images with proper attributes
- Creating ordered and unordered lists
- Structuring HTML documents properly
- Using Git and GitHub for version control
